This project uses Monte Carlo simulations and Python-based statistical modeling to analyze the potential long-term effects of tenant-friendly housing policies.
Objectives
Model NYC rental market volatility
Test Mamdani-style rent-stabilization policies
Simulate 10,000 alternative futures using Monte Carlo
Compare risk reduction vs renter cost burden
Key Skills Represented
Economic modeling
Probability distributions
Policy risk analysis
Scenario simulation
Data visualization
Python (NumPy, Pandas, Matplotlib)
Files
notebooks/analysis.ipynb → Main simulation
data/ → Contains sample synthetic datasets
results/ → Graphs + model summaries
